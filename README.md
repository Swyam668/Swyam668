## 🧠 Featured Projects

### 1. AI Career & Salary Intelligence Platform

An end-to-end AI-powered career analytics platform that analyzes resumes and generates personalized career insights.

**What it does:**

* Parses resumes and extracts candidate profiles using NLP
* Recommends relevant job roles based on skills and profile match
* Performs skill-gap analysis for each recommended role
* Predicts salary using machine learning
* Explains salary predictions using SHAP
* Generates personalized career roadmaps using LLMs
* Provides a full-stack dashboard for profile, recommendations, salary insights, and roadmap

**Tech Stack:**

`Python` `FastAPI` `Scikit-learn` `Sentence Transformers` `SHAP` `Gemini API` `Next.js` `TypeScript` `Tailwind CSS` `Docker` `Hugging Face Spaces` `Vercel`

**ML Highlights:**

* Trained salary prediction model on a dataset of **250,000 records**
* Achieved **R² Score: 0.96**
* Achieved **MAE: ₹5,436**
* Integrated SHAP explainability to show how features impact salary predictions
* Built a modular backend pipeline for resume parsing, profile extraction, recommendation, salary prediction, and roadmap generation

### 2. StudyGPT | AI-Powered Learning Platform

A Retrieval-Augmented Generation based learning platform that helps students learn from uploaded study material.

**What it does:**

* Allows users to upload PDF study materials
* Extracts and chunks document content
* Generates embeddings using Gemini Embeddings
* Stores vectors using MongoDB Atlas Vector Search
* Performs semantic search over uploaded documents
* Uses LLM reranking to improve retrieved context quality
* Generates flashcards, quizzes, summaries, and concept explanations
* Uses LLM-as-a-Judge evaluation to improve answer quality

**Tech Stack:**

`React.js` `Node.js` `Express.js` `MongoDB Atlas Vector Search` `Gemini API` `RAG` `LLM Reranking` `JWT Authentication`

**Repository:**
https://github.com/Swyam668/StudyGPT

**Live Demo:**
https://study-gpt-two.vercel.app
